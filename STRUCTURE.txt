🚀 Quick Commands:

    make quickstart     # Start everything (first time)
    make start          # Start services
    make setup          # Setup OpenFGA
    make open           # Open frontend
    make test           # Test API
    make logs           # View logs
    make stop           # Stop services
    make clean          # Clean everything
