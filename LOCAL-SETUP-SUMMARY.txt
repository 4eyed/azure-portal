1. Start your day:
   $ make start-openfga      # Start OpenFGA (if not running)
   $ make dev                # Start Functions

2. Code:
   - Edit backend/MenuApi/MenuFunction.cs
   - Save file
   - Functions auto-reloads!

3. Test:
   $ make test

4. Debug (optional):
   $ code backend/MenuApi    # Open in VS Code
   # Press F5 to debug

5. End of day:
   # Ctrl+C to stop Functions
   $ docker-compose stop openfga
